# How To Learn

## Описание
Одностраничный проект, написанный с использованием HTML и CSS с целью закрепления пройденного материала в рамках обучения.

## Стек:
* HTML
* CSS

## Технологии
* Flexbox
* Позиционирование
* Iframes
* Наполнение страницы сторонним контентом с использованием API
* БЭМ-разметка
* Структура файлов Nested
* Псевдоэлементы
